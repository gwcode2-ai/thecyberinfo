    📖 Cyber Dictionary

    Learn common cybersecurity terms:

    1. Phishing: A cyber attack where someone pretends to be a trusted person or company to trick you into sharing personal information.

    2. Malware: Harmful software created to damage devices, steal information, or disrupt systems.

    3. Encryption: A process that converts information into a secure code to prevent unauthorized access.

    4. Two-Factor Authentication (2FA): An extra security step that requires a second form of verification besides your password.

    5. Password Manager: A tool that securely stores and manages passwords.

    6. Firewall: A security system that monitors and controls network traffic.
